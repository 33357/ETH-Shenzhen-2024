# ETH-Shenzhen-2024

## Hackathon 项目提交说明

在 `Hackathon` 文件夹下创建一个以您的项目命名的文件夹。

项目文件夹内创建一个`Readme.md` 文件，内含以下内容：

- 项目概述
- 核心功能（4-5 条关键核心要点）
- 代码仓库地址
- 团队成员（名字、社交地址、Github 地址）
- Deck (PPT) 地址，也可直接放在此文件夹下
- 项目功能演示录屏（可选）
